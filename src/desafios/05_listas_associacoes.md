Em duplas, desenvolva um modelo UML e um código Java para cada um dos seguintes cenários:

1. Modele um sistema de **Projeto de Software** com `Projeto`, `Tarefa` e `Funcionario`. Um projeto tem várias tarefas, e cada tarefa pode ser atribuída a vários funcionários.
2. Modele um sistema de **Loja de Roupas** com `Departamento`, `Produto`, `Cliente` e `Carrinho`. Um cliente pode ter vários carrinhos e cada carrinho pode conter vários produtos.
